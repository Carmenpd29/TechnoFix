## Acceso y Seguridad
- Inicio de sesión: 
    - Accede con tu usuario y contraseña. Solo los usuarios autenticados pueden acceder a las funciones internas del sistema. 
    - Si es la primera vez, deberás registrarte. Después, el administrador tendrá que asignarte un rol para poder acceder.    

- Protección de rutas: 
    Si no has iniciado sesión, no podrás acceder a ninguna vista privada ni por URL directa. Serás redireccionado al login.

- Persistencia de sesión: 
    Tu sesión permanece activa, aunque recargues la página o cierres el navegador, salvo que cierres sesión manualmente.

- Roles: Existen 3 tipos de roles:
    - Administrador: Puede acceder y gestionar todas las secciones de la aplicación, incluyendo la administración de usuarios y la configuración de empresa. Puede crear, editar y eliminar clientes, reparaciones, productos y usuarios (excepto a sí mismo como admin principal).
    - Encargado: Puede crear y editar clientes, reparaciones y productos, así como realizar ventas. No puede eliminar ningún registro.
    - Empleado: Puede consultar, crear y editar clientes, reparaciones y ventas. No puede eliminar ningún registro.

- Menú principal: 
    - Acceso rápido a Clientes, Reparaciones, TPV (Caja), Usuarios y Configuración. 
    - El menú se adapta a tu rol y dispositivo. 
    - Podrás cambiar tu contraseña en tu nombre de usuario, visualizando debajo tu rol asignado.

## Recomendaciones y solución de problemas

	- Si detecta actividad sospechosa, errores de seguridad o accesos no autorizados, envíe un informe inmediato al administrador con capturas y pasos para reproducir el problema.


## Recomendaciones y solución de problemas
- Si la operación tarda más de 10s, recargue y vuelva a intentarlo.
- Verifique que el teléfono no esté duplicado al crear un cliente.
- Para problemas de autenticación contacte al administrador.
