# TechnoFix

Aplicación web para la gestión de reparaciones, clientes, ventas y usuarios en talleres técnicos.

## Instalación

1. Clona el repositorio:
	```sh
	git clone https://github.com/Carmenpd29/TechnoFix.git
	```
2. Instala las dependencias:
	```sh
	npm install
	npm install bootstrap
	npm install @supabase/supabase-js html2canvas jspdf jspdf-autotable styled-components react-icons express
	```
3. Ejecuta en modo desarrollo:
	```sh
	npm run dev
	```
4. Compila para producción:
	```sh
	npm run build
	```

## Tecnologías
- React 19
- Vite
- Bootstrap 5
- Supabase
