# Manual de Usuario TechnoFix

## 1. Introducción
TechnoFix es una plataforma web para la gestión integral de clientes, reparaciones y ventas en talleres técnicos.

## 2. Acceso y Seguridad
- **Inicio de sesión:** Accede con tu usuario y contraseña. Solo los usuarios autenticados pueden acceder a las funciones internas del sistema.
- **Protección de rutas:** Si no has iniciado sesión, no podrás acceder a ninguna vista privada ni por URL directa.
- **Persistencia de sesión:** Tu sesión permanece activa aunque recargues la página o cierres el navegador, salvo que cierres sesión manualmente.
- **Roles:** Solo los administradores pueden acceder a la gestión de usuarios y configuración de empresa.
- **Menú principal:** Acceso rápido a Clientes, Reparaciones, TPV (Caja), Usuarios y Configuración. El menú se adapta a tu rol y dispositivo.

## 3. Gestión de Clientes
- Buscar, añadir, modificar y eliminar clientes.
- Visualizar información y generar PDF de ficha de cliente.

## 4. Reparaciones
- Añadir nueva reparación vinculada a un cliente.
- Modificar y consultar reparaciones existentes.
- Imprimir o exportar ficha de reparación en PDF.

## 5. Terminal Punto de Venta (TPV)
- Añadir productos a la venta por código o nombre.
- Seleccionar cliente para la venta.
- Realizar cobros y gestionar líneas de venta.

## 6. Usuarios
- Registrar nuevos usuarios (solo administradores).
- Modificar tus propios datos o los de otros usuarios (según permisos y rol).
- El usuario administrador no puede ser editado ni eliminado desde la lista de usuarios.

## 7. Consejos de Uso
- Utiliza el buscador para encontrar clientes y productos rápidamente.
- Revisa los datos antes de confirmar ventas o reparaciones.
- Exporta fichas en PDF para entregar al cliente.
- Si ves el mensaje "Cargando..." tras iniciar sesión, espera unos segundos; el sistema verifica tu autenticación y permisos.

## 8. Personalización y Soporte
- Puedes personalizar el logo y nombre de empresa desde la sección Empresa (solo administradores).
- Para dudas técnicas, contacta con el administrador del sistema.
