# Manual de Usuario TechnoFix

## 1. Introducción
TechnoFix es una plataforma web para la gestión integral de clientes, reparaciones y ventas en talleres técnicos.

## 2. Acceso y Navegación
- **Inicio de sesión:** Accede con tu usuario y contraseña.
- **Menú principal:** Acceso rápido a Clientes, Reparaciones, TPV (Caja), Usuarios y Configuración.

## 3. Gestión de Clientes
- Buscar, añadir, modificar y eliminar clientes.
- Visualizar información y generar PDF de ficha de cliente.

## 4. Reparaciones
- Añadir nueva reparación vinculada a un cliente.
- Modificar y consultar reparaciones existentes.
- Imprimir o exportar ficha de reparación en PDF.

## 5. Terminal Punto de Venta (TPV)
- Añadir productos a la venta por código o nombre.
- Seleccionar cliente para la venta.
- Realizar cobros y gestionar líneas de venta.

## 6. Usuarios
- Registrar nuevos usuarios.
- Modificar datos propios o de otros usuarios (según permisos).

## 7. Consejos de Uso
- Utiliza el buscador para encontrar clientes y productos rápidamente.
- Revisa los datos antes de confirmar ventas o reparaciones.
- Exporta fichas en PDF para entregar al cliente.

## 8. Soporte
- Para dudas técnicas, contacta con el administrador del sistema.
